# Linux.do 抽奖自动回复扩展

Chrome/Edge MV3 扩展：监控 `linux.do` 最新帖子中的“抽奖”标签话题，自动回复并记录已回复历史。

## 安装（解压加载）
1. 打开 `chrome://extensions`。
2. 开启 **开发者模式**。
3. 点击 **加载已解压的扩展程序**，选择该目录：`linuxdo-browser-extension`。

## 使用
1. 打开 `https://linux.do/` 并确保已登录。
2. 在页面右侧打开扩展面板，开启 **自动回复** 开关。
3. 扩展会定时扫描 `/latest`，命中“抽奖”标签且可回复时自动回复一次。

## 说明
- 默认扫描间隔：30 秒（遇到 429 会退避）。
- 仅对每个话题回复一次，并同步你的历史回复记录。
- 回复成功会触发系统通知（带节流）。
- 面板支持收起/展开，状态保存在 `chrome.storage.local`。
