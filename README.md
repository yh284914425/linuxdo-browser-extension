# Linux.do 自动浏览扩展

Chrome/Edge MV3 扩展：从 `/latest` 顺序访问 linux.do 帖子，并随机停留。支持暂停/继续与断点续跑。

## 安装（解压加载）
1. 打开 `chrome://extensions`。
2. 开启 **开发者模式**。
3. 点击 **加载已解压的扩展程序**，选择该目录：`linuxdo-browser-extension`。

## 使用
1. 打开 `https://linux.do/latest` 并确保已登录。
2. 点击页面右下角面板的 **开始**。
3. 扩展会在同一标签页内依次浏览帖子，目标数量可在面板里设置（默认 1000，上限 1000）。

## 说明
- 默认随机停留范围：1–3 秒。
- 进度保存于 `chrome.storage.local`，刷新或重启后可继续。
- 若 JSON 接口被阻断，会自动回退为页面滚动抓取。
- 面板支持「重新开始」，会清空进度并重新从 `/latest` 构建队列。
